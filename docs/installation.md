# Wireless Adapter Installation

## Load with ESPHome

Once the ESPHome image is compiled, load it using the USB-C connector.

## Connect to Indoor Unit and Wireless Receiver

Remove power from the indoor unit (typically be removing power from the outdoor unit).

Remove the indoor unit's panel in order to access the CN105 connector. For my indoor units, instructions for removing the panel can be found in section "10. Disassembly Instructions" of the [Service Manual for the MSZ-GLxxNA and MSY-GLxxNA Indoor Units](https://www.mitsubishitechinfo.ca/sites/default/files/SH_MS%28Y%29%28Z%29-GL09_24NA_OBH732D.pdf).

If connected, unplug the wireless receiver from the indoor unit's CN105 connector

Attach the enclosure bottom to the wall with drywall anchors.

Connect the indoor unit's CN105 connector to the wireless adapter's IU connecter. For this, a cable is needed. I chose a [50cm, same direction, PAP-05V-S cable](https://www.aliexpress.us/item/3256810525305677.html). I found that a [20cm, same direction, PAP-05V-S cable](https://www.aliexpress.us/item/3256804560686405.html) was too short.

If present, connect the wireless receiver cable to the wireless adapter's WR connector.

Attach the PCBA to the enclosure bottom using four white nylon, 10mm long M3 standoffs.

Attach the enclosure top to the enclosure bottom using four white nylon, 15mm long, flat countersunk head M3 screws.

Replace the indoor unit's panel.

Apply power to the indoor unit.
